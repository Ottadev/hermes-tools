DocSplitter

Divide automaticamente un documento PDF o DOCX in più file .txt separati, usando i titoli in grassetto come separatori.

1
Carica un file .pdf o .docx tramite drag&drop o clic.
2
Analisi automatica dei grassetti: ogni testo in bold all'inizio di un paragrafo diventa un nuovo file.
3
Anteprima dei file che verranno generati, con nome e contenuto.
4
Scarica lo ZIP con tutti i file .txt e un indice riassuntivo.
Elaborazione 100% locale — nessun dato lascia il browser
