DocJoiner

Unisce selezionando i file da uno .zip generato da DocSplitter in un unico file .txt o .md.

1
Carica lo ZIP prodotto da DocSplitter tramite drag&drop o clic.
2
Seleziona i file che vuoi includere nel contesto — spunta cartelle o file singoli.
3
Scegli il formato di output: TXT (separatori) o MD (Markdown con code block).
4
Scarica il file unico pronto per essere usato come contesto.
